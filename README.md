## 📚 Learning Modules

### Module 1 — SDLC & STLC
> Software Development Life Cycle & Software Testing Life Cycle

- Understanding Waterfall & Agile methodologies
- Requirements analysis & test planning
- Test case design and execution
- Defect reporting and tracking

### Module 1.2 — Test Design Techniques
> Black-box testing methodologies

- **Equivalence Partitioning** — Dividing input data into valid/invalid groups
- **Boundary Value Analysis** — Testing edge cases at boundaries
- **Decision Table Testing** — Systematic approach for complex business rules
- **State Transition Testing** — Testing system behavior across states
- **Cause-Effect Graphing** — Visual technique for input combinations
